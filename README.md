## Installation

> Clone Repository

```
git clone https://github.com/SVTeamJ/DeepBlue.git

```

<br>

> Set .env in the docker folder


```
TZ=Asia/Seoul
MYSQL_USER=
MYSQL_ROOT_PASSWORD=
MYSQL_PASSWORD=
MYSQL_HOST=db
MYSQL_PORT=3306
MYSQL_DATABASE=
INSTA_APP_ID=
AWS_ACCESS_KEY_ID=
AWS_SECRET_KEY=
AWS_S3_BUCKET_NAME=
INSTA_APP_SECRET_ID=

```

<br>

> Run deployment env.


```
cd docker/frontend-web/
yarn install
cd ..
docker-compose up --build


```
